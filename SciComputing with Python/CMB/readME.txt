Universe evolution simulation based on the Cosmic microwave background data from the WMAP.

Author:
Evgeni Todorov
e.b.todorov@gmail.com

Instructions:
Running the cmb.py file will result in a simulation based on the 9th year release of WMAP data. The png source can be changed from the settings.py file, together with the simulation resoultion parameters. 

Dependencies:
Python v2.7 (including PIL)
Numpy v1.8
Pygame v1.9.2

Credits:
CMB data pictures credit to NASA/WMAP Science Team
Retrieved on 03.June.2014 from http://map.gsfc.nasa.gov/
PNG files are converted to black and white in RGBA color scheme with the indexing kept; black corresponds to blue, i.e. cold.
white corresponds to blue, i.e. hot.

All other content was generated for the purpose of this simulation
